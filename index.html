📂 데이터 관리
💾
JSON 내보내기
전체 데이터(부품+기록)를 JSON 파일로 저장합니다. 다른 PC에서 가져오기로 복원 가능합니다.
📂
JSON 가져오기
이전에 내보낸 JSON 파일을 불러옵니다. 기존 데이터에 병합하거나 덮어쓸 수 있습니다.
📊
CSV 내보내기
엑셀·구글 시트에서 열 수 있는 CSV 파일로 내보냅니다. 부품 목록과 입출고 기록을 포함합니다.
📥
CSV 가져오기
CSV 파일에서 부품 목록을 가져옵니다. 열 순서: 메이커,모델,연식,품번,품명,사용위치,입고가,출고가,최소재고,보관위치
📄 JSON 또는 CSV 파일을 여기에 드래그 & 드롭
💡 사용 팁
• 백업: 정기적으로 JSON 내보내기로 백업하세요
• 다른 PC로 이동: JSON 내보내기 → 다른 PC에서 JSON 가져오기
• 엑셀 연동: CSV 내보내기 후 엑셀에서 열어 보고서 작성
⚠ 위험 구역
전체 데이터 초기화
부품 목록 및 입출고 기록을 모두 삭제합니다. 되돌릴 수 없습니다.
